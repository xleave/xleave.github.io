PACER 多传感器健康监测与可解释故障诊断系统

基于多模态传感器（LiDAR/Camera）的健康状态评估原型系统，集成了离线重放引擎、特征提取及端到端故障归因诊断看板。

GitHub 仓库

项目详情

项目概述

为了保障自动驾驶及移动机器人系统的运行安全，传感器健康监测（Sensor Health Monitoring）是实现 Responsible AI 理念的重要物理防线。本项目开发了一套离线优先（Offline-first）的传感器多模态健康诊断系统，能够在运行时或录制包重放期间，精准检测传感器失效并定位根因。

核心功能与技术实现

1. 离线优先的重放引擎与传感器抽象

  • SensorFrame 抽象：设计了统一的传感器帧元数据结构 SensorFrame，实现了对图像、激光雷达点云（LiDAR point-cloud）以及车辆定位时间戳的精准对齐。
  • 重放框架：支持 nuScenes 和 CADC 数据集的包文件快速解析，无需高性能物理硬件即可完成传感器诊断流程演练。

2. 多维度健康特征提取 (Feature Engineering)

  • 图像视觉健康指标：提取模糊度（Laplacian Variance）、异常曝光与过饱和度、边缘密度等健康指标，实时发现镜头污渍、起雾或物理阻挡。
  • 三维点云退化评估：计算点云密度分布、最大有效距离、BEV (鸟瞰图) 占用栅格及遮挡范围，检测暴雨、大雾或物理遮蔽引起的探测效能衰减。

3. 可解释的多源故障融合与归因算法

  • 协同评估矩阵：构建了“双向不一致性（Pairwise Disagreement）”分析策略，将单一传感器异常分数与时序特征融合，精准计算出故障评分、共享事件评分及传感器退化状态。
  • 人机协同标注看板：基于 FastAPI 和原生 JS 构建轻量化交互看板，可视化展示时间线故障波形与多模态证据链，支持运维工程师进行二次确认和异常阈值的手动校准。
