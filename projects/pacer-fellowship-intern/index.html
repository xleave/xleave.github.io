PACER：多模态自动驾驶多传感器数据监测研究

在 Bristol Robotics Laboratory 开展的 PACER 项目学术研究，设计了多模态数据离线健康诊断算子，并在故障注入环境下评估了 Responsible AI 鲁棒性。

GitHub 仓库

项目详情

研究背景

伴随着自动驾驶与具身机器人的规模化落地，保证其环境感知系统在各类恶劣边界条件（如突降大雨、镜头起雾、LiDAR 故障）下的行为可靠性，是“负责任的具身智能 (Responsible AI in Robotics)”的核心议题。本研究依托英国布里斯托机器人实验室 (Bristol Robotics Laboratory) 的设备平台，主要研究多模态传感器在运行时的健康特征评估与故障归因模型。

核心研究成果

1. 多传感器特征工程设计 (Feature Engineering)

  • 多维度信号降级量化：实现了一套实时健康检测算子：
    • Camera (视觉频道)：评估实时模糊度（基于拉普拉斯方差）、局部曝光不均、高频边缘丢失以及直方图偏移（判断起雾/物理污渍遮挡）。
    • LiDAR (激光信道)：评估点云全局分布密度、最大回波距离分布、特定垂直角能量丢失以及 BEV (鸟瞰图) 空间占用退化。

2. 时序不一致故障融合算法

  • 故障共识归因：设计基于多传感器联合不一致性（Pairwise Disagreement）的故障辨识算法。当 Camera 图像退化（例如雨水模糊）时，模型结合 LiDAR 在相同锥形区域内的探测数据对比，计算出实时的 fault score 与 shared event score，自动判别到底是特定“Camera 硬件退化”还是全局的“突降大雾环境改变”。
  • 人机共识对齐：为非专家级操作员提供可视化时间线，高亮显示退化发生的点，为长周期的有监督根因分析（Root-cause Diagnosis）提供精准的数据标记。

3. 故障注入与鲁棒性验证

  • 故障注入系统：研发了自动故障注入系统（Fault-injection system），可在 nuScenes 重放数据流中动态屏蔽特定的点云扇区或对图像添加不同强度的噪声、伪影。
  • 性能测量：实验表明，该健康诊断算法在模拟发生感知退化后能快速触发报警，且在雨天等自然噪点干扰下的误报率（False Positive Rate）极低，展示出了高鲁棒性。

研发进展

2026/05/01

作为研究实习生加入布里斯托机器人实验室 (BRL) 的 PACER 课题组，开展自动驾驶可靠性研究。

2026/06/05

基于 Python 及 C++ 设计 Camera-LiDAR 的特征健康评估算子，包括图像模糊度、BEV 占用率与点云密度等。

2026/06/15

开发故障注入实验模块，通过模拟传感器信号缺失和退化，定量评估监控管道的响应延迟和误报率。
