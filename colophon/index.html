关于本站 / Colophon

本站是一个基于 Astro 框架与 Material Design 3 设计规范构建的个人静态博客站点，注重极简美学与工程化实践。

核心框架与内容管理

Astro 4 静态站点生成

采用 Astro 静态生成框架，配合部分岛屿（Islands）架构交互组件，大幅减少客户端 JavaScript 的体积，提供极致响应与加载体验。

Material Design 3 设计系统

遵循 Material 3 设计规范，基于 HSL 动态调色盘实现色彩计算，并提供全局统一样式的圆角、动态标高以及自适应强调色配置。

Decap CMS 静态后台管理

集成 Decap CMS 静态管理后台。支持本地离线编辑与 GitHub OAuth 安全鉴权，实现私有源码仓直接向公开静态站进行双库提交与协同发布。

Pagefind 离线检索与拼音匹配 搜索引擎

全站内容集成 Pagefind 本地索引引擎。打包时自动分析文档生成轻量索引，支持客户端秒级模糊检索，并独创支持针对中文的拼音首字母缩写检索匹配，无需 any 服务端 API 支持。

智能主题与个性化外观

日出日落智能主题跟随 智能感知

集成天文学公式，基于浏览器 Geolocation 物理经纬度与儒略日算法，精确获取访客当地的日出、日落时间，实现早晚主题的平滑自动流转。

毛玻璃与强调色主题编辑器 外观构建

侧滑式主题抽屉允许用户实时调节卡片透明度、毛玻璃模糊半径以及主强调色相，配置本地持久化并解决闪烁（FOUC）问题。

动态 Canvas 背景主题舱 沉浸动效

内置单例 Canvas 渲染器，支持连线粒子、数字雨（鼠标滑动打乱）、重力星空（鼠标引力吸引）及流动极光四种沉浸式动态背景切换与自动销毁。

多重背景肤色生态 (Multi-Skin) 预设背景

提供默认明亮与暗色沉浸皮肤，并设计了复古护眼沙纸（Sepia Cozy Paper）以及为移动设备 OLED 屏幕优化的纯黑模式（OLED Absolute Black），保护视力同时兼顾省电。

极客功能与开发游乐场

全局命令行控制台 (Ctrl+/) 命令行工具

基于终端指令风格设计，可通过键盘快捷键一键唤醒。支持 help、dark/light、play/pause、next 以及 go 等命令行指令，快捷控制全站状态。

交互式代码沙箱 (Playground) 实时运行

文章正文内置代码沙箱运行环境。支持 HTML/CSS/JS 代码一键就地生成沙盒 Iframe 并渲染运行，重定向 console 输出，将技术博文变为可调试游乐场。

音频频谱 Canvas 动态波形 音频可视化

利用 Web Audio API 建立全局单例 AudioContext 驱动，连接塞壬唱片 API 并在跨页面切换（View Transitions）时保持播放不间断，结合 Canvas 绘制动态频段跳动声波。

足迹地图与阅读辅助

IP 属地足迹轨迹地图 轨迹归档

自动收集全站文章的地理坐标（Geo）和位置（Location）属性，在 SVG 世界地图上动态绘制网格线、闪烁雷达点以及连接弧线，支持与侧栏双向交互联动。

稍后阅读本地书签夹 数据持久化

集成右侧侧滑书签抽屉，读者可一键收藏文章，数据自动存入 LocalStorage，支持跨页面即时更新、快捷跳转及一键清除。

书籍级 PDF 导出与页边距规范 阅读辅助

专为 `@media print` 优化的无噪书籍排版风格，隐藏侧栏、页眉页脚与音乐播放器，重塑高对比度字体与细边框，带给读者沉浸的打印/PDF导出体验。

关于我页足迹热力贡献日历 动态热力

关于我页面嵌入了 GitHub 风格的足迹与活动日历墙，展示过去一年（365天）的活跃度。日历格子的色彩透明度及底色联动当前主题色相（HSL），随强调色滑块拖拽实时变幻。

极致细节与微交互设计

自定义极细主题滚动条

适配全站可滚动区域与目录（TOC）卡片，采用 4px 极细、轨道背景完全透明的滚动条设计，随主题自适应前景色，解决粗糙系统默认滚动条带来的视觉遮挡。

页面背景滚动锁定 (Body Scroll Lock)

在唤起命令行控制台、搜索框、主题抽屉或书签抽屉等全局模态悬浮窗时，自动锁定页面 <body> 的滚动，避免后台页面同步滚动的交互冲突。

局部目录 (TOC) 样式穿透与恢复

利用 :global() 样式作用域穿透，解决由于局部目录目录控制按钮动态生成导致的组件 scoped 样式失效问题，确保其在跨页面过渡时可折叠与重新唤醒。

基础设施与部署

持续集成与交付 (CI/CD)

使用 GitHub Actions 自动化流水线。在代码推送后，自动触发安全与合规质检（build-check）、死链链路验证（check-links）、WebP 图像格式无损压缩替换等构建流水线。

静态资源与分布式托管

全部构建产物及 Pagefind 本地离线检索索引托管于 GitHub Pages，依托全球 CDN 分发加速，在 100% 零外部 CDN 第三方库依赖下实现秒级极速首屏响应。

SEO 强化与 RSS 订阅

支持针对 Pagefind 搜索引擎优化的拼音首字母检索，并自动生成 Sitemap 网站地图和 RSS 订阅源，无缝对接各类主流 RSS 阅读器。
