关于本站 / Colophon

本站是一个基于 Astro 框架与 Material Design 3 设计规范构建的个人静态博客站点，结合具身智能交互工具箱与极致工程实践。

核心框架与内容管理

Astro 静态生成 核心框架

采用 Astro 静态生成框架，配合岛屿（Islands）架构交互组件，大幅减少客户端 JavaScript 体积，提供极致响应与加载体验。

Material Design 3 设计系统

遵循 Material 3 设计规范，基于 HSL 动态调色盘实现色彩计算，提供全局统一样式的圆角、动态标高以及自适应强调色配置。

Pagefind 离线检索与拼音匹配 搜索引擎

全站内容集成 Pagefind 本地索引引擎。打包时自动生成轻量索引，支持客户端秒级模糊检索与针对中文的拼音首字母缩写匹配。

智能主题与个性化外观

日出日落智能主题跟随 智能感知

集成天文学公式，基于浏览器 Geolocation 物理经纬度与儒略日算法，精确获取访客当地日出、日落时间，实现早晚主题平滑自动流转。

多重背景肤色生态 (Multi-Skin) 预设背景

提供默认明亮与暗色沉浸皮肤，并设计了复古护眼沙纸（Sepia Cozy Paper）以及为移动设备 OLED 屏幕优化的纯黑模式（OLED Absolute Black）。

动态 Canvas 背景主题舱 沉浸动效

内置单例 Canvas 渲染器，支持连线粒子、数字雨、重力星空及流动极光四种沉浸式动态背景切换与自动资源销毁。

具身智能与交互式工程工具箱

机械臂逆运动学与 2D/3D 姿态

包含 2-DOF/3-DOF 逆运动学求解器（代数法与 CCD 迭代算法）、SO(3) 旋转矩阵线框投影、万向节死锁（Gimbal Lock）奇异性预警及连杆动力学仿真。

A* / Dijkstra 寻路热力图

支持随机迷宫生成、基于 OKLCH 色彩空间的节点拓展深度热力梯度与拓展耗时/路径步数实时统计。

学术论文极速闪阅 (RSVP Reader)

使用 3 列 CSS Grid 无抖动 ORP（最佳识别点）居中对齐，支持学术短语确定性分词、上下文实时高亮预览与全功能键盘快捷键。

代码与算法 Split 逐行对比器

支持左右分栏代码与算法配置重构对比，自动标记新增（绿底）与删除（红底）行，提升重构记录的阅读效率。

手绘批注系统与 Web Audio 触觉反馈

忒修斯之船多重手绘批注系统

集成了 Caveat、Kalam 与 Patrick Hand 三种手写字体，支持桌面端外侧留白边缘批注卡片（Margin Notes Layer）与折角书签动画。

Web Audio 物理触觉音效

零外部音效文件加载开销（0KB），利用 Web Audio API 动态合成微弱机械点击与按键触觉反馈，支持 Cmd+K 面板快速开关。

目录与滚动条 SVG 手绘连接线条

在侧栏 Floating Scroll-Map 节点悬停时，自动绘制动态延伸的手绘 SVG 贝塞尔曲线，精确连接节点与正文对应目标。

基础设施与自动化部署

持续集成与交付 (CI/CD)

使用 GitHub Actions 自动化流水线。代码推送后自动触发安全与合规质检、死链验证、WebP 图像格式压缩替换等构建步骤。

静态资源与分布式托管

全部构建产物托管于 GitHub Pages，依托全球 CDN 分发加速，在零外部第三方库依赖下实现秒级首屏响应。

SEO 强化与 RSS 订阅

支持针对 Pagefind 搜索引擎优化的拼音首字母检索，并自动生成 Sitemap 网站地图和 RSS 订阅源。
