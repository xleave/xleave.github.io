阻抗控制与力-位混合控制在精密操作中的原理与选型

#阻抗控制#力控制#位置控制#机械臂#柔顺控制

机械臂与环境发生物理接触时，纯位置控制可能产生过大的接触力，导致工件或机械臂损坏。柔顺控制（Compliant Control）使机械臂在接触时表现出弹性行为，安全地适应环境约束。阻抗控制和力-位混合控制是两种主流的柔顺控制策略。

1. 位置控制的局限

纯位置控制将机械臂末端驱动到目标位置，无论接触力有多大：

τ=Kp(qd−q)+Kd(q˙d−q˙)\tau = K_p(q_d - q) + K_d(\dot{q}_d - \dot{q})τ=Kp​(qd​−q)+Kd​(q˙​d​−q˙​)

当末端与刚性环境接触时，微小的位置误差会产生极大的接触力（因为刚性接触的刚度接近无穷大）。这在以下场景中是不可接受的：

  • 装配（如将轴插入孔中）。
  • 打磨和抛光（需要恒定接触力）。
  • 与人协作（安全要求限制接触力）。

2. 阻抗控制（Impedance Control）

2.1 核心思想

阻抗控制不直接控制力或位置，而是控制末端执行器与环境之间的动态关系——即阻抗（Impedance）。

期望的末端行为被建模为一个质量-弹簧-阻尼系统：

Mde¨+Dde˙+Kde=FextM_d \ddot{e} + D_d \dot{e} + K_d e = F_{\text{ext}}Md​e¨+Dd​e˙+Kd​e=Fext​

其中：

  • e=x−xde = x - x_de=x−xd​ 是末端位置偏差
  • Md,Dd,KdM_d, D_d, K_dMd​,Dd​,Kd​ 是期望的惯量、阻尼和刚度矩阵
  • FextF_{\text{ext}}Fext​ 是环境施加的外力

2.2 物理直觉

阻抗控制使末端执行器表现得像一个虚拟弹簧：

  • 无接触时：末端被弹簧拉向目标位置 xdx_dxd​（类似位置控制）。
  • 有接触时：外力压缩虚拟弹簧，末端偏离目标但限制了接触力。

Fcontact≈Kd⋅ΔxF_{\text{contact}} \approx K_d \cdot \Delta xFcontact​≈Kd​⋅Δx

刚度 KdK_dKd​ 越大，末端越”硬”（更像位置控制）；KdK_dKd​ 越小，末端越”软”（更顺从外力）。

2.3 参数选择

参数物理含义选择依据
KdK_dKd​（刚度）对位置偏差的恢复力任务精度要求：精密 → 高；安全 → 低
DdD_dDd​（阻尼）对速度的阻力通常设为临界阻尼 Dd=2KdMdD_d = 2\sqrt{K_d M_d}Dd​=2Kd​Md​​
MdM_dMd​（惯量）虚拟质量通常设为较小值以获得快速响应

2.4 笛卡尔空间阻抗控制

在笛卡尔空间中实现阻抗控制：

τ=JT(q)[Kd(xd−x)+Dd(x˙d−x˙)]+g(q)\tau = J^T(q) \left[ K_d(x_d - x) + D_d(\dot{x}_d - \dot{x}) \right] + g(q)τ=JT(q)[Kd​(xd​−x)+Dd​(x˙d​−x˙)]+g(q)

其中 JJJ 是雅可比矩阵，g(q)g(q)g(q) 是重力补偿项。

3. 力-位混合控制（Hybrid Force/Position Control）

3.1 核心思想

力-位混合控制（Raibert & Craig, 1981）在笛卡尔空间中将任务分解为位置控制方向和力控制方向：

  • 沿自由运动方向：执行位置控制。
  • 沿约束方向：执行力控制。

3.2 选择矩阵

使用选择矩阵 SSS 和 Sˉ=I−S\bar{S} = I - SSˉ=I−S 将任务空间分为力控方向和位控方向：

τ=JT[S⋅Fd+Sˉ⋅Kp(xd−x)]+g(q)\tau = J^T \left[ S \cdot F_d + \bar{S} \cdot K_p(x_d - x) \right] + g(q)τ=JT[S⋅Fd​+Sˉ⋅Kp​(xd​−x)]+g(q)

其中 FdF_dFd​ 是期望力，SSS 是对角矩阵，对角元素为 0 或 1。

示例：沿桌面擦拭

S = diag(0, 0, 1, 0, 0, 0)
// z 方向力控（保持恒定下压力），xy 方向位控（控制擦拭路径）

3.3 物理直觉

力-位混合控制的直觉是任务自然约束的对偶性：

  • 如果末端在某方向上可以自由运动（无约束），则该方向适合位置控制。
  • 如果末端在某方向上被环境约束（如桌面），则该方向适合力控制。

4. 阻抗控制 vs. 力-位混合控制

维度阻抗控制力-位混合控制
控制对象力-位移关系（阻抗）力和位置（分别在不同方向）
力传感器不严格必需必需（力控方向需要力反馈）
方向切换各方向行为一致需要预先定义力控/位控方向
稳定性本质稳定（被动性）需要仔细设计以保证稳定
接触/非接触切换平滑过渡非接触→接触切换可能不稳定
精确力控间接（通过刚度和偏差）直接控制接触力

5. 选型指南

任务场景推荐方法原因
安全人机协作阻抗控制本质安全，无需力传感器
恒力打磨/抛光力-位混合控制需要精确控制法向力
插拔装配阻抗控制需要在多方向柔顺适应
沿表面滑动力-位混合控制法向力控 + 切向位控
VLA 策略对接阻抗控制VLA 输出位姿目标，阻抗控制作为底层
环境刚度未知阻抗控制刚度可在线调整，适应未知环境

6. 与学习策略的集成

在 VLA/RL 驱动的机器人系统中，阻抗控制器通常作为底层安全层：

VLA 策略 → 输出目标位姿 x_d（低频，5-20 Hz）
                ↓
阻抗控制器 → 计算关节力矩 τ（高频，1 kHz）
                ↓
电机驱动 → 执行力矩

策略只需输出目标位姿，阻抗控制器自动处理接触适应和力限制，降低了策略学习的难度。

可变刚度策略

更高级的方法是让策略不仅输出目标位姿，还输出阻抗参数（Kd,DdK_d, D_dKd​,Dd​），使机器人在需要精确定位时表现为高刚度，在需要柔顺适应时表现为低刚度。

总结

阻抗控制和力-位混合控制是机械臂柔顺控制的两种基本策略。阻抗控制通过控制力-位移关系实现隐式的柔顺行为，本质稳定且适用范围广；力-位混合控制通过在不同方向分别控制力和位置实现精确的接触力控制。在 VLA 驱动的机器人系统中，阻抗控制作为底层安全层与高层学习策略解耦，是当前最常见的集成方式。
