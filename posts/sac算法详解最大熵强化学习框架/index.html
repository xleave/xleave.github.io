SAC（Soft Actor-Critic）算法详解：最大熵强化学习框架

#SAC#强化学习#最大熵#连续控制#Actor-Critic
无障碍阅读:

Soft Actor-Critic（SAC）是一种基于最大熵（Maximum Entropy）原理的 off-policy Actor-Critic 算法，由 Haarnoja et al. 在 2018 年提出。它在连续动作空间的控制任务中展现了优异的样本效率和训练稳定性，广泛应用于机器人运动控制和操作任务。

1. 最大熵强化学习

1.1 标准 RL 目标 vs. 最大熵目标

标准 RL 的目标是最大化累积期望回报：

Jstd(π)=Eπ[∑t=0Tγtr(st,at)]J_{\text{std}}(\pi) = \mathbb{E}_\pi \left[ \sum_{t=0}^{T} \gamma^t r(s_t, a_t) \right]Jstd​(π)=Eπ​[∑t=0T​γtr(st​,at​)]

最大熵 RL 在此基础上加入策略熵的正则项：

JMaxEnt(π)=Eπ[∑t=0Tγt(r(st,at)+αH(π(⋅∣st)))]J_{\text{MaxEnt}}(\pi) = \mathbb{E}_\pi \left[ \sum_{t=0}^{T} \gamma^t \left( r(s_t, a_t) + \alpha \mathcal{H}(\pi(\cdot|s_t)) \right) \right]JMaxEnt​(π)=Eπ​[∑t=0T​γt(r(st​,at​)+αH(π(⋅∣st​)))]

其中 α>0\alpha > 0α>0 是温度参数，H(π(⋅∣s))=−Ea∼π[log⁡π(a∣s)]\mathcal{H}(\pi(\cdot|s)) = -\mathbb{E}_{a \sim \pi}[\log \pi(a|s)]H(π(⋅∣s))=−Ea∼π​[logπ(a∣s)] 是策略在状态 sss 下的熵。

1.2 最大熵的动机

加入熵正则项带来三个关键优势：

优势说明
鼓励探索高熵策略更随机，有助于发现更优的行为模式
鲁棒性策略不过度依赖特定动作，对环境扰动更鲁棒
多模态捕获当多个动作同样优秀时，策略保持多样性而非坍缩到单一模式

2. Soft Bellman 方程

在最大熵框架下，值函数的定义发生变化：

Soft State Value：

V(s)=Ea∼π[Q(s,a)−αlog⁡π(a∣s)]V(s) = \mathbb{E}_{a \sim \pi} \left[ Q(s, a) - \alpha \log \pi(a|s) \right]V(s)=Ea∼π​[Q(s,a)−αlogπ(a∣s)]

Soft Q-Value（Soft Bellman 方程）：

Q(s,a)=r(s,a)+γEs′∼p[V(s′)]Q(s, a) = r(s, a) + \gamma \mathbb{E}_{s' \sim p} \left[ V(s') \right]Q(s,a)=r(s,a)+γEs′∼p​[V(s′)]

将 VVV 代入 QQQ：

Q(s,a)=r(s,a)+γEs′[Ea′∼π[Q(s′,a′)−αlog⁡π(a′∣s′)]]Q(s, a) = r(s, a) + \gamma \mathbb{E}_{s'} \left[ \mathbb{E}_{a' \sim \pi} \left[ Q(s', a') - \alpha \log \pi(a'|s') \right] \right]Q(s,a)=r(s,a)+γEs′​[Ea′∼π​[Q(s′,a′)−αlogπ(a′∣s′)]]

3. SAC 的三组网络

SAC 使用三组可学习的网络（原始版本还包含 V 网络，后来简化）：

3.1 策略网络（Actor）πϕ\pi_\phiπϕ​

输出动作的概率分布。对于连续动作空间，通常使用可重参数化的高斯分布：

πϕ(a∣s)=N(μϕ(s),σϕ(s)2)\pi_\phi(a|s) = \mathcal{N}(\mu_\phi(s), \sigma_\phi(s)^2)πϕ​(a∣s)=N(μϕ​(s),σϕ​(s)2)

采样时使用重参数化技巧（Reparameterization Trick）：

a=tanh⁡(μϕ(s)+σϕ(s)⊙ϵ),ϵ∼N(0,I)a = \tanh(\mu_\phi(s) + \sigma_\phi(s) \odot \epsilon), \quad \epsilon \sim \mathcal{N}(0, I)a=tanh(μϕ​(s)+σϕ​(s)⊙ϵ),ϵ∼N(0,I)

tanh 将动作限制在 [−1,1][-1, 1][−1,1] 范围内。

3.2 双 Q 网络（Critic）Qθ1,Qθ2Q_{\theta_1}, Q_{\theta_2}Qθ1​​,Qθ2​​

SAC 使用两个独立的 Q 网络，取其最小值来估计 Q 值：

Qmin(s,a)=min⁡(Qθ1(s,a),Qθ2(s,a))Q_{\text{min}}(s, a) = \min(Q_{\theta_1}(s, a), Q_{\theta_2}(s, a))Qmin​(s,a)=min(Qθ1​​(s,a),Qθ2​​(s,a))

双 Q 的动机：单个 Q 网络容易高估 Q 值（Overestimation Bias），取最小值提供了更保守的估计，类似于 TD3 中的 Clipped Double Q-learning。

3.3 目标网络（Target Network）

Q 网络的目标值使用目标网络 QθˉiQ_{\bar{\theta}_i}Qθˉi​​ 计算，通过指数移动平均（EMA）软更新：

θˉi←τθi+(1−τ)θˉi,τ≪1\bar{\theta}_i \leftarrow \tau \theta_i + (1 - \tau) \bar{\theta}_i, \quad \tau \ll 1θˉi​←τθi​+(1−τ)θˉi​,τ≪1

4. 训练过程

4.1 Q 网络更新

最小化 Soft Bellman 残差：

LQ(θi)=E(s,a,r,s′)∼B[(Qθi(s,a)−Q^(s,a))2]\mathcal{L}_Q(\theta_i) = \mathbb{E}_{(s,a,r,s') \sim \mathcal{B}} \left[ \left( Q_{\theta_i}(s, a) - \hat{Q}(s, a) \right)^2 \right]LQ​(θi​)=E(s,a,r,s′)∼B​[(Qθi​​(s,a)−Q^​(s,a))2]

其中目标 Q 值：

Q^(s,a)=r+γ(min⁡j=1,2Qθˉj(s′,a~′)−αlog⁡πϕ(a~′∣s′))\hat{Q}(s, a) = r + \gamma \left( \min_{j=1,2} Q_{\bar{\theta}_j}(s', \tilde{a}') - \alpha \log \pi_\phi(\tilde{a}'|s') \right)Q^​(s,a)=r+γ(minj=1,2​Qθˉj​​(s′,a~′)−αlogπϕ​(a~′∣s′))

a~′∼πϕ(⋅∣s′)\tilde{a}' \sim \pi_\phi(\cdot|s')a~′∼πϕ​(⋅∣s′) 是从当前策略采样的下一步动作。

4.2 策略更新

最大化 Soft Q 值减去熵惩罚：

Lπ(ϕ)=Es∼B[Ea∼πϕ[αlog⁡πϕ(a∣s)−Qmin(s,a)]]\mathcal{L}_\pi(\phi) = \mathbb{E}_{s \sim \mathcal{B}} \left[ \mathbb{E}_{a \sim \pi_\phi} \left[ \alpha \log \pi_\phi(a|s) - Q_{\text{min}}(s, a) \right] \right]Lπ​(ϕ)=Es∼B​[Ea∼πϕ​​[αlogπϕ​(a∣s)−Qmin​(s,a)]]

通过重参数化技巧使梯度可以反向传播通过采样操作。

4.3 自动温度调节

温度参数 α\alphaα 控制探索与利用的平衡。SAC v2 引入了自动温度调节：

Lα=Ea∼π[−α(log⁡π(a∣s)+Hˉ)]\mathcal{L}_\alpha = \mathbb{E}_{a \sim \pi} \left[ -\alpha (\log \pi(a|s) + \bar{\mathcal{H}}) \right]Lα​=Ea∼π​[−α(logπ(a∣s)+Hˉ)]

其中 Hˉ\bar{\mathcal{H}}Hˉ 是目标熵，通常设为 −dim⁡(A)-\dim(\mathcal{A})−dim(A)（动作空间维度的负数）。

直觉：当策略的熵低于目标熵时，α\alphaα 增大以鼓励更多探索；当熵高于目标时，α\alphaα 减小以允许更多利用。

5. 完整训练算法

初始化: Q_θ1, Q_θ2, π_φ, 目标网络 Q_θ̄1, Q_θ̄2, 回放缓冲区 B, 温度 α

for each environment step:
    a ~ π_φ(·|s)
    s', r = environment.step(a)
    B.store(s, a, r, s')

for each gradient step:
    sample mini-batch (s, a, r, s') from B

    // 更新 Q 网络
    ã' ~ π_φ(·|s')
    Q̂ = r + γ(min Q_θ̄(s', ã') - α log π_φ(ã'|s'))
    更新 θ1, θ2 最小化 (Q_θi(s,a) - Q̂)²

    // 更新策略
    ã ~ π_φ(·|s)  (重参数化)
    更新 φ 最小化 α log π_φ(ã|s) - min Q_θ(s, ã)

    // 更新温度
    更新 α 最小化 -α(log π_φ(ã|s) + H̄)

    // 软更新目标网络
    θ̄i ← τθi + (1-τ)θ̄i

6. SAC vs. PPO 对比

维度SACPPO
数据使用Off-policy（回放缓冲区）On-policy（用后即弃）
样本效率高（数据可复用）低（数据不可复用）
训练稳定性高（最大熵正则化）高（裁剪机制）
探索策略隐式（最大熵）隐式（熵正则化）
连续动作空间原生支持支持
离散动作空间需要修改（SAC-Discrete）原生支持
并行环境不依赖通常依赖大规模并行
超参数敏感度较低（自动温度）中等

选型建议

  • 样本成本高（如真实机器人交互）：选择 SAC（样本效率更高）。
  • 仿真环境可大规模并行：选择 PPO（并行吞吐量更高）。
  • 连续控制任务：两者均可，SAC 通常在单环境设置下更快收敛。
  • 离散动作空间：优先选择 PPO。

总结

SAC 通过最大熵框架将探索动机编码进优化目标，配合双 Q 网络缓解高估偏差和自动温度调节平衡探索-利用，在连续控制任务中实现了高样本效率和训练稳定性。其 off-policy 特性使其能复用历史数据，特别适合样本成本高昂的机器人学习场景。
